# FitAble Feature Improvements Plan

## Overview

FitAble is an AI-powered fitness tracking web app (SvelteKit + Svelte 5 + Tailwind + PostgreSQL/Drizzle). This plan focuses on **improving existing features** based on user direction.

---

## Feature 1: Voice Coach Enhancements

### Voice Coach Code Polish

- **File:** `src/lib/components/voice-coach/voice-coach.ts:159,165`
- **Issue:** Uses `==` instead of `===` for comparisons
- **Fix:** Replace with strict equality

### Voice Coach Robustness

- **File:** `src/lib/components/voice-coach/voice-coach.ts`
- **Issue:** `recentUtteredWord` string comparison is fragile - typo risk ("watch-form" vs "watch_form")
- **Fix:** Use constants or enum for command keys

### Voice Recognition Pattern Collision

- **File:** `src/lib/components/voice-recognition/voice-recognition.ts:63-71`
- **Issue:** "let's go" and "let's stop" could collide since both patterns might match
- **Fix:** Check more specific patterns before general ones

## Feature 2: Workout UI/UX Polish

### Exercise Selection Mobile Grid

- **File:** `src/routes/(protected)/workout/+page.svelte:377-390`
- **Issue:** Grid uses `grid-cols-4` - too small touch targets on mobile
- **Fix:** Change to responsive grid `grid-cols-2 sm:grid-cols-4`

### Exercise Buttons Accessibility

- **File:** `src/routes/(protected)/workout/+page.svelte:377-390`
- **Issue:** Exercise buttons lack `aria-label` for screen readers
- **Fix:** Add descriptive `aria-label` with full exercise name

### Loading/Empty State for Workout

- **File:** `src/routes/(protected)/workout/+page.svelte`
- **Issue:** No loading skeleton or empty state before workout starts
- **Fix:** Add loading spinner or skeleton for camera initialization

### Hardcoded Exercise State Colors

- **File:** `src/routes/(protected)/workout/+page.svelte:641-648`
- **Issue:** Exercise badges use hardcoded colors (`bg-green-500`, etc.) instead of design tokens
- **Fix:** Use CSS variables or Tailwind semantic colors

### Safety Alert Overlay Polish

- **File:** `src/routes/(protected)/workout/+page.svelte:597-618`

- **Fix:** Use semantic color classes or CSS variables

### Exercise Guide Consistency

- **File:** `src/routes/(protected)/workout/+page.svelte:401-419`

### Video Element Accessibility

- **File:** `src/lib/components/pose-detector/pose-detector.svelte:465`

### Workout Post-Completion Feedback
